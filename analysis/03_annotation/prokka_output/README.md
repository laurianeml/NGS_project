# Genome annotation

This directory contains the genome annotation results.

## Tool

- Prokka

## Input

- PacBio assembled genome (FASTA)

## Output

- GFF file (used for RNA-seq read counting)
- GenBank file (.gbk)
- Protein sequences (.faa)
- Nucleotide sequences of predicted genes (.ffn)

## Purpose

Genome annotation provides gene models and functional information required
for RNA-seq mapping and expression analysis.
